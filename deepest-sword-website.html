Deepest Sword

A tiny game about a hero with a ridiculously long sword

Game Description

Deepest Sword is a tiny game about a hero with a ridiculously long sword made in 48 hours for the GMTK Game Jam 2021. The game was created by the team Cosmic Adventure Squad.

This game jam entry is based on the theme "Joined Together" and explores this concept through the connection between the hero and their impractical weapon.

Controls

  • Move - A and D
  • Sword - Mouse buttons or left and right arrows
  • Respawn - R
  • Full reset - 0 (zero)
